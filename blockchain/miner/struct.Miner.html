pub struct Miner {
    pub id: u32,
    pub network: HashSet<(u32, String)>,
    pub blocks: Vec<Block>,
    pub sockip: String,
    pub wallets: HashSet<(u32, String)>,
    pub payload: Vec<String>,
    pub current_block_id: u32,
}

This struct is used to represent a Miner in our Blockchain Network *id - the ID of our Miner, should be unique within the network *network - a HashSet containing the IDs and IP addresses of every Miner inside the network *blocks - a Vector containing the Blockchain *sockip - the IP address on which our Miner will listen for incoming transactions *wallets - a HashSet containing the IDs and IP addresses of every wallets binded to it *payload - TBD *current_block_id - TBD

pub struct Wallet {
    pub id: u32,

    pub socket: String,

This struct represent the wallets *id - an integer than should be unique to each wallet within a miner's wallet list *miner - the IP address of the miner to which the wallet is binded *socket - the IP address on which the wallet listens for incoming messages
